# lost-in-translation

A browser based geography game to learn Chinese country names. Usually on Chinese airlines, maps are displayed with location names in Chinese. Most are just transliterations so to amuse myself I would try to guess the cities (my idea to add this mode in the future) or country names.

Go to ken-diep.github.io/lost-in-translation to try it out. Works best on desktop but should be fine on mobile too.

Uses the beautiful Stamen watercolor map and Stamen terrain labels for country names. It has to be hosted on a local server to display the maps; if re-hosting, you will need to register on Stadia Maps and whitelist the domain. Credit also to the Leaflet js library and Leaflet providers repo.
